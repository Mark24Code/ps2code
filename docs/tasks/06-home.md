# Task 06 — 主界面(+号 / 拖拽 / 去重 / 路由)

## 目标
应用首页:醒目入口导入 PSD,展示项目列表,支持路由回主界面。

## 功能
- 列表上方一个**巨大的 + 号**卡片:
  - 支持拖拽本地 .psd 到其上导入。
  - 点击弹系统文件选择器(`dialog.showOpenDialog`,`filters: [{ name: 'PSD', extensions: ['psd'] }]`)。
- 导入即建 project(Task 02),psd 文件名作项目名;按绝对路径去重,已存在则直接打开对应项目记录。
- 项目列表:读 sqlite,展示名称/路径/时间,点击进入项目(Task 07 对话界面)。
- 顶部路由栏:随时回主界面(react-router-dom)。

## 视觉风格
- 高级感:黑白灰 + 商务蓝主色,**不要渐变色**。
- 卡片、留白、清晰层级;+ 号大而克制。

## 交互细节
- 拖拽高亮反馈;非 .psd 文件拒绝并提示。
- 拖入的路径经 IPC 交主进程处理(渲染进程 File 只能拿到 name,需 `webUtils.getPathForFile` 或 drop 事件 path 取绝对路径)。

## 验收
- 拖入 `design-drafts/a签到.psd` → 新建项目并进入。
- 再次拖入同文件 → 打开既有项目,不新增记录。

## 依赖
- Task 02(db)、Task 03(psd 解析,用于进入项目时读图层)
